Command Line Interfaces (CLIs)

A command-line interface (CLI) is a method of interacting with software via commands entered in a terminal or command prompt. Each command is formatted as a line of text.

Prompt Engines

Oh My Posh

Oh My Posh is a fast prompt engine for any shell, written in Go, with a rich set of themes and customizations.

  • GitHub Repository

Nerd Fonts

Nerd Fonts is a collection of patched fonts that include a wide range of glyphs, making them ideal for use in command-line interfaces and terminal applications.
