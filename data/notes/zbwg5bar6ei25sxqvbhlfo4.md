
A command-line interface (CLI) is a method of interacting with software via commands entered in a terminal or command prompt.
Each command is formatted as a line of text.

## Prompt Engines

### Oh My Posh

[Oh My Posh](https://ohmyposh.dev/) is a fast prompt engine for any shell, written in Go, with a rich set of themes and customizations.

- [GitHub Repository](https://github.com/JanDeDobbeleer/oh-my-posh)

## Nerd Fonts

[Nerd Fonts](https://www.nerdfonts.com/) is a collection of patched fonts that include a wide range of glyphs, making them ideal for use in command-line interfaces and terminal applications.

- [Fira Code](https://github.com/tonsky/FiraCode)
- [Meslo](https://github.com/andreberg/Meslo-Font)
